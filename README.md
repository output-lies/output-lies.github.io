# output lies

Source for **[output-lies.github.io](https://output-lies.github.io/)**: an educational site about how a diff, a terminal, or a pasted command can render untrusted bytes so that what you see is not what runs, and the safe tools that catch it.

Static HTML and CSS, no build step. Served by GitHub Pages from `main`.

## Pages

- `/` hub
- `/tools/` safe terminal text tools ("cat lies too")
- `/paste/` copy-paste and hidden-text attacks ("paste lies")
- The adversarial git-diff corpus and its site live in the [git-diffs-lie](https://github.com/output-lies/git-diffs-lie) repo, served at [/git-diffs-lie/](https://output-lies.github.io/git-diffs-lie/).

`output-lies.png` is the organization logo.

Built with AI assistance. Every demonstration is benign and for education only.
